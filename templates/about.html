PEMBUATAN WEB PROFILE

Web Profile ini dibuat, Yang fungsinya untuk menjelaskan tentang Cloud Computing. Yang dimana Cloud Computing adalah Cloud computing adalah penyediaan layanan komputasi melalui internet, yang mencakup penyimpanan data, server, basis data, jaringan, perangkat lunak, dan analitika. Layanan ini disediakan secara on-demand, memungkinkan pengguna untuk mengakses sumber daya IT tanpa memiliki atau mengelola infrastruktur fisik. Pada materi ini kita ditugaskan berupa :

1. Menyusun prototype pembuatan Web Profile dari segi HTML dan CSS yang dibutuhkan sesuai aturan yang diberikan.

2. Menginstall Flask dan VirtualEnv sesuai dengan langkah - langkah yang diberikan. Dan penginstalan harus sesuai folder direktori

3. Membuat App.py yang berisi pemograman python sebagai app route dengan metode Flask, Sebagai penghubung antara web dan flask Python.

4. Testing Web yang sudah terhubung dengan Flask dengan cara running App.py dan masuk ke web Localhost.

5. Deploy Web Profile melalui Platform Microsoft Azure. Dengan metode App Service.
